# PWA-main

Это сайт на `Next.js` с подключением к `Supabase`.

Сейчас проект сделан как система работы с заказами:
- список заказов;
- страница отдельного заказа;
- вход по `Supabase Auth`;
- хранение данных в `Supabase`;
- деплой можно делать на `Vercel`.

## Что где лежит

- `app/` - страницы сайта
- `components/` - визуальные блоки интерфейса
- `lib/` - вспомогательный код
- `lib/auth/` - логика, связанная с пользователем и профилем
- `lib/orders/` - типы, константы, правила доступа и утилиты по заказам
- `public/` - картинки и статические файлы

## Что важно настроить

Проекту нужны переменные окружения.

Создай файл `.env.local` и добавь туда:

```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```

Для удобства рядом есть шаблон: [.env.example](/Users/timon/Documents/Codex/2026-04-24-new-chat/project/PWA-main/.env.example)

## Как запустить локально

1. Установи зависимости:

```bash
npm install
```

2. Запусти проект:

```bash
npm run dev
```

3. Открой в браузере:

```text
http://localhost:3000
```

## Деплой на Vercel

В `Vercel` нужно добавить те же переменные:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

Без них сайт не сможет подключиться к базе.
